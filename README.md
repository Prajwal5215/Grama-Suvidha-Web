# Grama Suvidha Web

Grama Suvidha Web is a web-based digital notice board designed to improve transparency and accessibility in village development projects. The platform allows citizens to view information about ongoing and completed government works in their village through a simple and user-friendly interface.

## Features

* View village development projects and their details.
* Display project budget information.
* Show contractor details.
* Track project progress and completion status.
* View expected completion dates.
* Access site photos of ongoing works.
* Provide citizen feedback functionality.
* Support multiple languages for better accessibility.

## Purpose

The objective of this project is to increase transparency in village administration by making public work information easily available to citizens and reducing the communication gap between local authorities and residents.
